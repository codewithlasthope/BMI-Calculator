# 🧮 BMI Calculator

A simple, responsive, and user-friendly BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript. This web application allows users to quickly calculate their BMI and determine their weight category based on standard BMI ranges.

## 🌐 Live Demo

**Try it here:** https://codewithlasthope.github.io/BMI-Calculator/

---

## ✨ Features

* Calculate BMI instantly using height and weight inputs
* Displays BMI value with proper classification
* Categorizes results into:

  * Underweight
  * Normal Weight
  * Overweight
  * Obese
* Input validation for a better user experience
* Responsive design for desktop and mobile devices
* Clean and modern user interface

---

## 🧮 BMI Formula

BMI is calculated using the standard formula:

BMI = Weight (kg) / Height² (m²)

This formula is widely used to estimate body fat based on height and weight.
